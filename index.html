Projeto para estoque de um mercado

Com esse programa é possível gerenciar o estoque, realizar cadastro de produtos e ele também avisa se a quantidade de determinado produto no estoque está abaixo ou acima do esperado.

Projeto idade do eleitor

Esse programa verifica se um eleitor está apto a votar de acordo com a sua idade. Ao informar a idade, ele dispara uma mensagem dizendo se essa pessoa pode ou não votar, e também se o voto é obrigatório.

Projeto para cadastro de carros

Esse programa realiza o cadastro do carro no site, usando dados como modelo, placa, ano, cor, marca e outros.
